1.以#开头的语句称为预处理器指令
#include <stdio.h>
2.书写注释是一个良好的编程习惯  单行注释以"//"开头  多行注释以"/*"开头以"*/"结尾
3.变量名的首字母不能是数字  变量名不能包含除_以外的任何特殊字符  不可以使用保留字
4.变量
   int num = 1 //声明并初始化
   int num1
   num1 = 2 //先声明在初始化
5.ASCII码 空格 32  A~Z 65~90 a~z 97~122   
   记忆方法  大写和小写之间差一个空格
6.char 的声明只能是单个字符而不是字符串
7.%只能算整形
8.scanf("%...",&...);     //scanf函数输入时不能规定精度
9.getchar  获取字符  格式为: a = getchar(...);
   putchar  输出字符  格式为: putchar('字符串'或直接在括号里写变量);
10.fflush(stdin);	//清空缓冲区
11.%m.nlf                //m是控制宽度  n是控制小数点后位数
12.宏常量的定义格式: #define 标识符 字符串
13.const常量只能在定义时赋初值,格式是: const double a = 3;
     //其他类推
14.强转暂时只知道 printf("a = %d\n",(int)a);  这样可以转
15.退格 \b  //即删除前一个字    回车 \r   //即删除一行前面所有字
16.两个scanf函数在一起可能把回车键输入第二个scanf函数,防止的方法有getcahr()读取缓冲区的回车键或在scanf()前加空格,即scanf(" %c",&c);	//%c前加了空格
